# Trifid Cipher

## Mathematical Definition

Extension of Bifid using a $3 \times 3 \times 3$ cube (27 characters) with trifid fractionation over $(layer, row, col)$ coordinates.

## Python Implementation

See `cipherops/ciphers/fractionated.py::trifid`.

## Dataset

Validated examples: `datasets/fingerprinted/trifid-keyword/data.jsonl`
